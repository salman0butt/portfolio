## 💅 Design System

### Typography
```css
Heading 1:  5xl - 7xl (72-96px)
Heading 2:  3xl - 4xl (36-48px)
Body:       lg - xl (18-20px)
Small:      sm - base (14-16px)

Font:       Inter (clean sans-serif)
Weight:     400 (regular), 600 (semibold), 700 (bold)
```

### Spacing
```css
Section padding:    py-20 (80px)
Card padding:       p-8 (32px)
Button padding:     px-8 py-4 (32px 16px)
Gap between elements: gap-4 to gap-12
```

### Borders & Radius
```css
Border radius:  12px (buttons), 16px (cards), 999px (badges)
Border width:   1px (cards), 2px (buttons/icons)
Border color:   slate-200 (light), slate-700 (dark)
```
